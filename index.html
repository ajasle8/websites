Run a Simple Website on a Chromebook with Linux Enabled

Step 1: Enable Linux (if not already done)

  • Open Settings on your Chromebook.
  • Go to Advanced → Developers.
  • Enable Linux Development Environment (if it isn’t already enabled). This will install Linux and open a terminal.

Step 2: Install Python

  1. Open the Linux terminal on your Chromebook.
  2. Check if Python is installed: 
    python3 --version
  3. If Python is not installed, install it using: 
    sudo apt update
sudo apt install python3

Step 3: Create Your Server Script

  1. In the terminal, create a new folder for your project: 
    mkdir my_website
cd my_website
  2. Create the simple_server.py file using a text editor like nano: 
    nano simple_server.py
  3. Copy and paste this code into the file: 
    import http.server
import socketserver

PORT = 8000

Handler = http.server.SimpleHTTPRequestHandler

with socketserver.TCPServer(("", PORT), Handler) as httpd:
    print(f"Serving at http://127.0.0.1:{PORT}")
    httpd.serve_forever()
  4. Save the file (press Ctrl+O, then Enter) and exit the editor (press Ctrl+X).

Step 4: Create Your HTML File

  1. Create an index.html file in the same folder: 
    nano index.html
  2. Add the following content: 
    <!DOCTYPE html>
<html>
<head>
    <title>Simple Website</title>
</head>
<body>
    <h1>Welcome to My Simple Website!</h1>
    <p>This is served using Python on a Chromebook.</p>
</body>
</html>
  3. Save and exit the editor.

Step 5: Run the Server

  1. Run the server: 
    python3 simple_server.py
  2. Open a web browser and go to: 
    http://127.0.0.1:8000
  3. You should see your website!

Bonus: Keep Your Server Running

  1. If you want the server to keep running even after closing the terminal, use nohup: 
    nohup python3 simple_server.py &
  2. This will run the server in the background. To stop it, find its process ID: 
    ps aux | grep simple_server.py
  3. And then kill it: 
    kill <PROCESS_ID>
